## Introduction

App CR is a modern task management API built with Express.js, Prisma ORM, and PostgreSQL. It provides a complete backend solution for managing users and tasks with JWT-based authentication.

## Quick Example

Create a new user and authenticate:

```bash
# Create a new user
curl -X POST http://localhost:3000/usuarios \
  -H "Content-Type: application/json" \
  -d '{
    "email": "user@example.com",
    "password": "securepassword"
  }'
```
